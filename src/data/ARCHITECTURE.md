# src/data — Static content

Static JSON content rendered by the page sections: `profile.json`,
`project.json`, `skills.json`, `xp.json`, `languages.json`.

Entries reference translations by key (e.g. `titleKey: "projects.skinbattle.title"`)
— when adding such a key, add the matching translation to **all 12**
`messages/*` files (see the `add-translations` skill).
